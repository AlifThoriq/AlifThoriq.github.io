Apa itu Web Design?

Web Design

Web design adalah istilah mengenai design visual yang diterapkan kepada media digital yaitu website, design yang ditunjukan untuk pengembangan dan styling informasi di internet untuk menyediakan fitur konsumen high-end dan memberikan kualitas estetika. Definisi yang ditawarkan memisahkan desain web dari pemrograman web, menekankan fitur fungsional dari sebuah situs web, serta desain posisi web sebagai semacam desain grafis.

Secara tidak langsung, website design ini fokus pada tampilan, tapi tidak akan menghilangkan fungsi utama design itu sendiri. Dengan tampilan Design yang bagus mampu memeberikan kenyamanan tambahan bagi pengunjung dan pembaca website.

UI/UX

Setelah kita membahas web designer, lantas apakah sama seorang web designer dan ui/ux designer? Mari kita bahas.

UI/UX designer merupakan orang yang menangani tampilan dari suatu produk, seperti aplikasi atau website. Dalam praktiknya, seorang UI/UX designer wajib memikirkan ekspektasi user dan memahaminya. Setelah itu, ekspektasi dihadirkan melalui produk yang ia desain. Di tahap akhir, ia mengevaluasi apakah produk yang sudah dibuat sudah memenuhi ekspektasi dan keinginan mereka.

Dari kedua penjelasan di atas kita dapat simpulkan bahwasanya seorang web designer fokus pada pengembangan website. Tidak hanya soal desain, ia juga harus menyesuaikan semua dengan bahasa pemrograman. Sementara itu, UI/UX designer bisa bekerja tidak hanya pada website, tetapi juga produk lain dari suatu perusahaan. Misalnya, aplikasi mobile, aplikasi website, dan lain sebagainya.

Tugas UI/UX

Pada tugas kali ini, kita diharuskan untuk membuat UI mentah terlebih dahulu sebelum kita membuat tampilan UI yang sebenarnya. Disini aku menggunakan tampilan web portofolio projek aku yang sudah aku buat dari SMA. Sebenarnya ada beberapa UI/UX yang dulu ketika SMA sudah aku buat code nya.

Oleh karena itu, ketika aku membuat gambar tampilan UI/UX di kertas, aku menggabungkan beberapa elemen JavaScript seperti animasi Slide, Light up ketika card di sentuh oleh kursor, dan beberapa fitur lainnya. Disini juga aku tidak lupa menambahkan beberapa code CSS supaya memudahkan aku dalam mengatur warna, layout, dan fitur yang akan ditampilkan di Website.
